# S7-Mini-Project 🚀

Welcome to **S7-Mini-Project**, a modern web application integrating a Next.js frontend with a backend powered by FastMCP and Starlette. The project demonstrates the use of CopilotKit to interact with MCP tools, aiming to build interactive, AI-powered user experiences.

---

## 📝 Introduction

**S7-Mini-Project** brings together the best of modern web technologies:
- **Next.js** for the frontend, offering a scalable and easy-to-use framework.
- **FastMCP** and **Starlette** for the backend, enabling efficient server-side communications.
- **CopilotKit** integration for seamless interaction with AI-powered tools.

This project is an ideal starting point for developers interested in full-stack development and building applications that leverage AI capabilities.

---

## ✨ Features

- **Next.js Frontend**: Fast, efficient, and feature-rich app structure.
- **CopilotKit Integration**: Connects your app with MCP server tools and AI services.
- **Tailwind CSS**: Modern, utility-first CSS styling.
- **Custom Manifest & PWA Setup**: Ready for mobile and desktop experiences.
- **API Routing**: Example endpoints for interacting with OpenAI and MCP.
- **TypeScript Support**: Ensures code reliability and maintainability.

## ⚙️ Installation

### Prerequisites

- **Node.js** (v18+ recommended)
- **npm** or **yarn**
- **Python 3.8+** (for backend)
- MCP server and dependencies

### 1. Clone the Repository

```bash
git clone https://github.com/Pranesh-2005/S7-Mini-Project.git
cd S7-Mini-Project
```

### 2. Setup Frontend

```bash
cd my-app
npm install
```

### 3. Setup Backend

Install Python dependencies (example):

```bash
pip install -r requirements.txt
```

Or, manually install:

```bash
pip install httpx starlette uvicorn fastmcp
```

---

## 🚀 Usage

### 1. Run the Backend

```bash
python ap.py
```
*Ensure any required environment variables are set (e.g., for OpenAI API keys).*

### 2. Run the Frontend

```bash
cd my-app
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.
